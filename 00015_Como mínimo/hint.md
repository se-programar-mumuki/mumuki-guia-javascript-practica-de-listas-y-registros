Para poder hacer esta nueva función probablemente te sirva tanto `minimo` como `gananciasDeBalancesPositivos`. :sunglasses:

`minimo` se utiliza como `maximo`:

```javascript
ム minimo([5, 8, 10, 42, 87, 776])
5
```
